# 쿠팡 파트너스 연동 가이드 🛒

## 📋 개요

쿠팡 파트너스는 **API 없이 제휴 링크로 작동**하는 간단한 수익화 방법입니다.
- 사용자가 링크를 클릭하고 구매하면 자동으로 커미션 적립
- 커미션율: 3-5% (카테고리별 상이)
- 정산: 월 1회 (최소 10,000원 이상)

---

## 🚀 설정 방법 (5분 완료)

### 1단계: 쿠팡 파트너스 가입

1. **가입 페이지 접속**
   ```
   https://partners.coupang.com
   ```

2. **회원가입**
   - 이메일 또는 소셜 로그인
   - 사업자 정보 입력 (개인/법인)
   - 블로그/앱 정보 입력

3. **심사 대기**
   - 보통 1-3일 소요
   - 승인 메일 수신

### 2단계: 파트너 ID 확인

1. **대시보드 접속**
   ```
   https://partners.coupang.com/dashboard
   ```

2. **파트너 ID 복사**
   - 우측 상단 프로필 클릭
   - "파트너 정보" 메뉴
   - 파트너 ID 복사 (예: AF1234567)

### 3단계: 환경 변수 설정

1. **`.env.local` 파일 생성** (프로젝트 루트)
   ```bash
   # .env.local
   NEXT_PUBLIC_COUPANG_PARTNER_ID=AF1234567
   ```

2. **Vercel/Railway 환경 변수 설정**
   - Vercel: Settings → Environment Variables
   - Railway: Variables 탭
   - 키: `NEXT_PUBLIC_COUPANG_PARTNER_ID`
   - 값: `AF1234567` (본인의 파트너 ID)

### 4단계: 테스트

1. **로컬 서버 재시작**
   ```bash
   npm run dev
   ```

2. **추천 결과 확인**
   - 음식 사진 촬영
   - 추천 결과에서 "쿠팡에서 구매하기" 버튼 확인
   - 버튼 클릭 시 쿠팡 페이지로 이동

3. **링크 확인**
   - URL에 `trcid=AF1234567` 포함 확인
   - 이 파라미터가 있어야 커미션 적립
